
# Web Snake Multiplayer (2 Players) + Voice + Fruit + Retry + Leaderboard

Features added in this version:
- Fruit visible & respawning
- Eating fruit increases score + grows tail
- Retry button to reset your snake without reloading
- Leaderboard sorted by score
- Player name selection
- WebRTC voice chat
- 2 players + spectators supported

Run:
```
npm install
node server.js
```
Open http://localhost:3000 in two browser windows.
